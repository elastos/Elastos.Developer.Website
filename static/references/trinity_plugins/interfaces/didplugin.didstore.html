synchronize

  • synchronize(storepass: string, onSuccess: function, onError?: function): any

  • This methods synchronizes a whole DID store content, from the DID sidechain, to the local device.

    • Every DID previously generated by this DID store (same mnemonic = same private identity) will be found on chain and re-created locally if it doesn't exist.
    • All DID documents and credentials on chain will also be restored locally.

    This method can be used in 2 different use cases:

    1. To restore a user's identity from scratch
    2. In case user uses the same identity on multiple devices, to synchronize data changed on chain by one app to the second app.

    NOTE: Only data previously saved on chain can be restore. This method cannot restore private credentials kept by the user on his device.

    Parameters

    • storepass: string
    • onSuccess: function
        • (): void

        • Returns void

    • Optional onError: function
        • (err: any): void

        • Parameters

          • err: any

          Returns void

    Returns any
