addStream

  • addStream(type: StreamType, options: Number, callbacks: StreamCallbacks, onSuccess: function, onError?: function): any

  • Add a new stream to session. Carrier stream supports several underlying transport mechanisms:

    • Plain/encrypted UDP data gram protocol
    • Plain/encrypted TCP like reliable stream protocol
    • Multiplexing over UDP
    • Multiplexing over TCP like reliable protocol Application can use options to specify the new stream mode. Multiplexing over UDP can not provide reliable transport.

    Parameters

    • type: StreamType

      The stream type defined in StreamType

    • options: Number

      The stream mode options. Options are constructed by a bitwise-inclusive OR of flags

    • callbacks: StreamCallbacks

      The stream callbacks.

    • onSuccess: function

      The function to call when success, the param is a Stream object: The new added carrier stream.

        • (stream: Stream): void

        • Parameters

          • stream: Stream

          Returns void

    • Optional onError: function

      The function to call when error, the param is a string. Or set to null.

        • (err: string): void

        • Parameters

          • err: string

          Returns void

    Returns any
