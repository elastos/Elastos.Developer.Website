Interface AuthenticationHandler

Interface that must be implemented by dApps when creating a hive client instance. It is used to receive an authentication challenge from a hive back-end and return a authentication response.

authenticationChallenge

  • authenticationChallenge(jwtToken: string): Promise<string>

  • Called whenever the hive back-end needs to authenticate this client. The received JWT token contains a nonce. The application must create a DID verifiable presentation that embeds a special app identification credential (usually generated by the Identity dapp) and the nonce. This presentation is then packaged as a response JWT, signed by the application instance DID, and returned to the back-end. The back-end will then issue an access token that can be used to all future vault API calls.

    Parameters

    • jwtToken: string

    Returns Promise<string>
