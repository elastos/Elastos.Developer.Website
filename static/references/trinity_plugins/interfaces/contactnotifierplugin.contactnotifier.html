acceptInvitation

  • acceptInvitation(invitationId: string): Promise<Contact>

  • Accepts an invitation sent by a remote peer. After accepting an invitation, a new contact is saved with his did and carrier addresses. After that, this contact can be resolved as a contact object from his did string.

    Parameters

    • invitationId: string

      Received invitation id.

    Returns Promise<Contact>

    The generated contact

getCarrierAddress

  • getCarrierAddress(): Promise<string>

  • Returns DID-session specific carrier address for the current user. This is the address that can be shared with future contacts so they can send invitation requests.

    Returns Promise<string>

    The currently active carrier address on which user can be reached by (future) contacts.

sendInvitation

  • sendInvitation(did: string, carrierAddress: string): any

  • Sends a contact request to a peer. This contact will receive a notification about this request and can choose to accept the invitation or not.

    In case the invitation is accepted, both peers become friends on carrier and in this contact notifier and can start sending remote notifications to each other.

    Use invitation accepted listener API to get informed of changes.

    Parameters

    • did: string

      Target contact DID.

    • carrierAddress: string

      Target carrier address. Usually shared privately or publicly by the future contact.

    Returns any
