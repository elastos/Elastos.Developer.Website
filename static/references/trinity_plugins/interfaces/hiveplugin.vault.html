Methods

getDatabase

  • getDatabase(): Database

  • Gives access to database features on this vault.

    Returns null in case we are accessing a vault that is not the current user's.

    Returns Database

getFiles

  • getFiles(): Files

  • Gives access to files features on this vault.

    Returns Files

getNodeVersion

  • getNodeVersion(): Promise<string>

  • Returns the deployment version for this vault's vault provider (node). This can be useful to maintain backward compatibility with older nodes.

    Returns Promise<string>

getPayment

  • getPayment(): Payment

  • Gives access to the payment module that allows paying for vault usage, check active subscriptions and quotas and more.

    Returns null in case we are accessing a vault that is not the current user's.

    Returns Payment

getScripting

  • getScripting(): Scripting

  • Gives access to all vault scripting features on this vault.

    Returns Scripting

getVaultOwnerDid

  • getVaultOwnerDid(): string

  • DID string of this vault owner

    Returns string

getVaultProviderAddress

  • getVaultProviderAddress(): string

  • provider address (carrier or http)

    Returns string

revokeAccessToken

  • revokeAccessToken(): Promise<void>

  • Revokes the existing access token linking the current user with the vault. Next operation on the vault after calling launches a new authentication flow in order to generate a new access token.

    Returns Promise<void>
