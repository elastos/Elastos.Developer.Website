Module HivePlugin

This is about Hive which provides the IPFS(InterPlanetary File System, a protocol and peer-to-peer network for storing and sharing data in a distributed file system)-based APIs.

Please use 'HivePlugin' as the plugin name in the manifest.json if you want to use this facility.

Usage:
declare let hiveManager: HivePlugin.HiveManager;
