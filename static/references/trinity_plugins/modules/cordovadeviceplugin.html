Module CordovaDevicePlugin

This plugin provides an API for getting the device's hardware and software information.

Please use 'Device' as the plugin name in the manifest.json if you want to use this facility.

Usage:
device.getInfo((info)=>{console.log(info.platform)}, (err)=>{});

Type aliases

DeviceInfo

DeviceInfo: object

The device information.

Type declaration

  • available: boolean

    Indicates that Cordova initialize successfully.

  • cordova: string

    Get the version of Cordova running on the device.

  • isVirtual: boolean

    Whether the device is running on a simulator.

  • manufacturer: string

    Get the device's manufacturer.

  • model: string

    The device.model returns the name of the device's model or product. The value is set by the device manufacturer and may be different across versions of the same product.

  • platform: string

    Get the device's operating system name.

  • serial: string

    Get the device hardware serial number.

  • uuid: string

    Get the device's Universally Unique Identifier (UUID).

  • version: string

    Get the operating system version.
