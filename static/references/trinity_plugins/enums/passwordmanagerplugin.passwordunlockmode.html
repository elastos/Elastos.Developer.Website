Enumeration PasswordUnlockMode

Mode defining how often the passwords database has to be unlocked in order to access application or user password info.

Index

Enumeration members

  • UNLOCK_EVERY_TIME
  • UNLOCK_FOR_A_WHILE

Enumeration members

UNLOCK_EVERY_TIME

UNLOCK_EVERY_TIME: = 1

Users have to provide their master password every time an application requests a password. This provides higher security in case the device is stolen, but this is less convenient for users.

UNLOCK_FOR_A_WHILE

UNLOCK_FOR_A_WHILE: = 0

After been unlocked once, password manager access is open during some time and until elastOS exits. Users don't have to provide their master password again during this time, and all apps can get their password information directly.
